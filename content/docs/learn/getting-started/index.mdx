Seyfert is a fantastic library focused on Discord and its API for bots. It offers support for scalable applications and provides unique utilities.

<Callout>
We recommend having a solid foundation in TypeScript before using Seyfert for better understanding and fluency.
</Callout>

## Installation

### Prerequisites

- [NodeJS](https://nodejs.org/) installed (v18 or higher)
- [TypeScript](https://www.typescriptlang.org/) installed (v4.9 or higher)
- NPM installed (or a similar NodeJS package manager)

Seyfert also supports [Deno](https://deno.land/) and [Bun](https://bun.sh/) starting from version **2.2.0**.

### Setup

```bash copy title="Installing..."
npm add seyfert
```
